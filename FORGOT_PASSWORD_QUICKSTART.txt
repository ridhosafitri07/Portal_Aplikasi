🚀 NEXT STEPS - SETUP GUIDE
═══════════════════════════════════════════════════════════════════════════════

### STEP 1: Database Migration
```bash
cd d:\laragon\www\portal_aplikasi
php spark migrate
```

Ini akan membuat table `password_resets` di database.

### STEP 2: Install & Run WhatsApp Bot
```bash
cd whatsapp-bot

# Option A: Manual install
npm install
npm start

# Option B: Quick start (Windows)
start.bat

# Option B: Quick start (Linux/Mac)
bash start.sh
```

### STEP 3: Login WhatsApp dengan QR Code

Ketika bot berjalan, Anda akan melihat QR Code di terminal:
```
╔════════════════════════════════════════╗
║    SCAN QR CODE DENGAN WHATSAPP BARU   ║
╚════════════════════════════════════════╝

████████████████████████████████████████████
█...banyak character...█
█████ QR CODE ██████████
```

**PENTING:**
- Gunakan WhatsApp di **smartphone/nomor BARU** (jangan WhatsApp sehari-hari)
- Buka WhatsApp → ⋮ (tiga titik) → Perangkat Tertaut → Tautkan Perangkat
- Scan QR Code dengan kamera WhatsApp
- Tunggu sampai terminal menampilkan: ✅ WhatsApp Bot READY!

### STEP 4: Test Fitur

1. Buka: http://localhost/portal_aplikasi/forgot-password
2. Masukkan username & nomor HP yang terdaftar
3. Cek WhatsApp - Anda akan menerima OTP
4. Ikuti flow sampai selesai

⚠️ PERHATIAN
═══════════════════════════════════════════════════════════════════════════════

1. **Bot harus berjalan** untuk fitur forgot password berfungsi
   - Jangan tutup terminal bot
   - Jika diclose, user tidak bisa menerima OTP

2. **QR Code hanya tampil sekali per sesi**
   - Bila QR Code hilang, buka `whatsapp-bot/.wwebjs_auth/`
   - Hapus folder tersebut untuk reset session
   - Jalankan `npm start` lagi

3. **Gunakan WhatsApp baru untuk bot**
   - Jangan pakai WhatsApp pribadi
   - Bot akan install sebagai "WhatsApp Web"
   - Smartphone harus online agar bot tetap connect

4. **Validasi nomor HP**
   - Format bisa: 08123456789 atau +6281234567
   - Bot otomatis format ke +62xxxx
   - Pastikan nomor valid & terdaftar di sistem
